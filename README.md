# AuroralisStar by Jack

Fork creativo inspirado en Eclipse/BSL, retocado para tener una identidad propia con auroras mas intensas, paleta morado/dorado, menu curado y presets separados para hardware distinto.

## Contenido

- `laptop-mx450/`
  Variante enfocada en rendimiento para MX450.
- `desktop-rtx4060/`
  Variante enfocada en calidad para RTX 4060.

## Perfiles principales

- `Auroralis MX450 (Objetivo 120 FPS)`
- `Auroralis Fast (MX450)`
- `Auroralis RTX 4060 (Calidad Alta)`
- `Auroralis Cinema (4060)`

## Controles recomendados del fork

Dentro del menu `AuroralisStar by Jack` puedes ajustar rapido:

- Auroras
- Niebla general
- Niebla por hora del dia
- Niebla de lluvia
- Niebla de cuevas
- Color grading
- Bloom y sharpening
- Direccion artistica

## Notas

- La version MX450 busca estabilidad y desactiva partes pesadas como shader grass por teselacion.
- La version RTX 4060 empuja mas el look cinematografico y las auroras.
- El idioma base del fork ya tiene una capa visible en espanol para el menu principal del shader.
