# AuroralisStar by Jack

Fork creativo con identidad propia, retocado para tener auroras mas intensas, paleta curada, temas atmosfericos y presets separados para hardware distinto.

## Creditos y base

AuroralisStar no parte de cero. Este fork se apoya sobre el trabajo de shaders previos y mantiene el credito correspondiente:

- Eclipse Shader, de Merlin1809:
  https://github.com/Merlin1809/Eclipse-Shader
- Bliss Shader, de X0nk:
  https://github.com/X0nk/Bliss-Shader
- Chocapic13, por la base historica citada por los proyectos upstream

La idea de este repo es construir una identidad nueva sobre esa linea, no borrar su origen.

## Contenido

- `laptop-mx450/`
  Variante enfocada en rendimiento para MX450.
- `desktop-rtx4060/`
  Variante enfocada en calidad para RTX 4060.

## Perfiles principales

- `Auroralis MX450 (Objetivo 120 FPS)`
- `Auroralis Fast (MX450)`
- `Auroralis RTX 4060 (Calidad Alta)`
- `Auroralis Cinema (4060)`

## Controles recomendados del fork

Dentro del menu `AuroralisStar by Jack` puedes ajustar rapido:

- Auroras
- Niebla general
- Niebla por hora del dia
- Niebla de lluvia
- Niebla de cuevas
- Color grading
- Bloom y sharpening
- Direccion artistica

## Notas

- La version MX450 busca estabilidad y desactiva partes pesadas como shader grass por teselacion.
- La version RTX 4060 empuja mas el look cinematografico y las auroras.
- El idioma base del fork ya tiene una capa visible en espanol para el menu principal del shader.
